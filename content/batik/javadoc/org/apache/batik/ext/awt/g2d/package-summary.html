Package org.apache.batik.ext.awt.g2d

Provides an API for subclassing the java.awt.Graphics2D class in order to translate Java 2D primitives into another graphic format.

See:
          Description

Class Summary
AbstractGraphics2D This extension of the java.awt.Graphics2D abstract class is still abstract, but it implements a lot of the Graphics2D method in a way that concrete implementations can reuse.
DefaultGraphics2D This concrete implementation of AbstractGraphics2D is a simple help to programmers to get started with their own implementation of Graphics2D.
GraphicContext Handles the attributes in a graphic context:
+ Composite
+ Font
+ Paint
+ Stroke
+ Clip
+ RenderingHints
+ AffineTransform
TransformStackElement Contains a description of an elementary transform stack element, such as a rotate or translate.
TransformType Enumeration for transformation types.
